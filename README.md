# vitals-review

A small two-page static site comparing two lab panels (Nov 2025 → Jun 2026) and a personalised 6-month action plan.

- **index.html** — overview: wins, in-progress, new flags, full marker table, and 6-month targets.
- **plan.html** — week-by-week 6-month action plan, diet & lifestyle playbooks.
- **style.css** — shared "Card Atlas" stylesheet (sage theme).

All personally identifying details have been removed. Built to be hosted on GitHub Pages (main / root).

> Educational self-care reference only — not medical advice. Interpret with a treating physician.
